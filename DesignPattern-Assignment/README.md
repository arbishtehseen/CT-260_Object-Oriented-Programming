This repository contains a comprehensive technical report and architectural analysis of Duolingo, 
exploring how it leverages industry-standard Object-Oriented Design Patterns to manage language
learning at scale.Developed as part of the Object-Oriented Programming (CT-260) course at NED University, 
this project identifies and implements 8 core GoF patterns including:
Creational: Abstract Factory (UI Rendering), Factory Method (Exercise Engine).
Structural: Composite (Course Hierarchy), Flyweight (Memory-efficient Tile Rendering).
Behavioral: Observer (Gamification & Rewards), Memento (Session Management), Strategy (Exercise Evaluation), 
and Chain of Responsibility (Validation Pipeline).
Included is a full C++ implementation demonstrating these patterns in a decoupled, extensible,
and professional codebase, alongside detailed UML Class Diagrams.
